# CS3502-P3
File System Project
by Dylan K.

# Features
Create, read, update, delete, rename (which incidentally can move files), and navigation abilities

# Prerequisites
No prerequisites provided you have Python 3.10.4 installed. Not tested on more modern versions; there is a small but nonzero chance of deprecation.

Some test files are included for fun.

# How to run
After installing Python, run `main.py`.
